# שימוש ב-Ollama (LLM מקומי)

## התקנה:

1. הורד והתקן Ollama מ: https://ollama.com/download
2. פתח CMD והרץ:
   ```
   ollama pull llama3.2
   ```
3. התקן את החבילה:
   ```
   pip install llama-index-llms-ollama
   ```

## שימוש:

המערכת כבר מוכנה לעבוד עם Ollama!
פשוט הרץ:
```
python app.py
```

## יתרונות:
✅ עובד לגמרי offline
✅ תשובות מנוסחות ויפות
✅ חינמי לחלוטין
✅ פרטיות מלאה - הכל על המחשב שלך

## חסרונות:
❌ צריך מחשב חזק (8GB RAM מינימום)
❌ תשובות יותר איטיות (5-10 שניות)
