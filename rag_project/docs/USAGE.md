# 🚀 מדריך שימוש מהיר

## התחלה מהירה

### 1. הרצת המערכת
```bash
python app.py
```

### 2. פתיחת הממשק
פתח דפדפן: `http://127.0.0.1:7860`

### 3. שאל שאלות
```
"מה הצבע העיקרי בדיזיין?"
"רשום את כל ההחלטות הטכניות"
"איך מתקינים את המערכת?"
```

## סוגי שאילתות

### שאילתות סמנטיות (Semantic)
שאלות פתוחות, הסברים:
- "מה ההחלטות הטכניות?"
- "הסבר את הארכיטקטורה"
- "איך עובד הניתוב?"

### שאילתות מובנות (Structured)
רשימות, סינונים:
- "רשום את כל ההחלטות"
- "הצג רק אזהרות קריטיות"
- "מה השינויים האחרונים?"

## פתרון בעיות מהיר

### תשובות לא רלוונטיות
1. בנה אינדקס מחדש: `python build_index.py`
2. הוסף יותר מסמכים ל-`mock_docs/`
3. שנה את `CHUNK_SIZE` ב-`config.py`

### תשובות איטיות
1. הקטן את `TOP_K` ב-workflow
2. השתמש ב-Ollama במקום Cohere
3. הפחת את `max_attempts`

### שגיאות API
1. בדוק מפתח: `python test_keys.py`
2. ודא חיבור לאינטרנט
3. בדוק מכסת API ב-Cohere
