📋 Workflow Components

Steps (@step decorator):
  • validate_input - Checks if query is valid (not empty, min 3 chars)
  • retrieve_nodes - Searches vector index for relevant documents
  • check_quality - Calculates confidence score (threshold: 0.2)
  • format_response - Formats final response with sources
Events:
  • StartEvent - Workflow entry point with user query
  • InputValidatedEvent - Query passed validation
  • NodesRetrievedEvent - Relevant documents found
  • QualityCheckedEvent - Quality assessment complete
  • StopEvent - Workflow exit with result or error
State Management:
  • Query string flows through all events
  • Nodes stored after retrieval
  • Confidence score calculated and passed
  • Context available across all steps
